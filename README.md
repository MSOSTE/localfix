# LocalFix

LocalFix is a Django web application for reporting neighborhood maintenance issues.

Residents can create reports, add photos, choose a location on a map, filter reports, and follow report status changes. Moderators can manage all reports and update their statuses.

## Features

- User registration, login and logout
- Cookie/session based authentication
- Two roles: user and moderator
- PostgreSQL database
- Latvian and English language switching
- Report CRUD: create, view, edit and delete
- Multiple photo upload and photo deletion
- Categories with translated names and map colors
- Status filtering and "My reports" filtering
- Pagination after 10 reports
- AJAX report filtering without full page reload
- Real-time report statistics with Server-Sent Events
- Address geocoding with OpenStreetMap Nominatim
- Interactive maps with Leaflet and OpenStreetMap
- Click on map to fill address and coordinates
- Custom 403 and 404 error pages
- Django admin panel for users, profiles, categories and reports
- Responsive navbar with mobile menu

## Roles

User:

- can create reports
- can edit and delete only own reports
- can view all reports
- can filter reports and use the map

Moderator:

- can edit and delete all reports
- can change report status
- can view all reports
- can use the same filters and map

Moderator role is assigned in the Django admin panel:

```text
Accounts -> Profiles -> role = moderator
```

The user also needs staff status only if they must log into the Django admin panel.

English pages use the `/en/` prefix.

## API and Dynamic Endpoints

```text
GET /lv/reports/
```

Returns the report list. When called with the `X-Requested-With: XMLHttpRequest` header, it returns filtered report cards as JSON for AJAX filtering.

```text
GET /lv/reports/live-stats/
```

Server-Sent Events endpoint. It streams live report statistics every 5 seconds.

Example event:

```text
data: {"total": 3, "new": 2, "in_progress": 1, "resolved": 0, "rejected": 0}
```

```text
GET /lv/reverse-geocode/?lat=56.949600&lng=24.105200
```

Returns an address for selected map coordinates.

## External APIs

The project uses free OpenStreetMap services:

- OpenStreetMap map tiles for map display
- Nominatim for address to coordinates
- Nominatim reverse geocoding for coordinates to address

No API key is required.

## Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a PostgreSQL database named `localfix`.

Create `.env` from `.env.example`:

```text
DJANGO_SECRET_KEY=change-me
DJANGO_DEBUG=1
DJANGO_ALLOWED_HOSTS=127.0.0.1,localhost
POSTGRES_DB=localfix
POSTGRES_USER=localfix
POSTGRES_PASSWORD=localfix
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_CONNECT_TIMEOUT=5
GEOCODER_USER_AGENT=LocalFix student project
GEOCODER_TIMEOUT=5
```

Apply migrations:

```bash
python manage.py migrate
```

Start the server:

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/lv/
```

## Admin Panel

Create an admin user:

```bash
python manage.py createsuperuser
```

Open:

```text
http://127.0.0.1:8000/lv/admin/
```

The admin panel is generated by Django and configured in the `admin.py` files.
